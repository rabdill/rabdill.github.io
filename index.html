Who I am

My name is Rich. I live in Minneapolis. I hang out with two cats, two dogs and a lovely woman named Lauren. My hobbies include genealogy and ruining home improvement projects. My favorite color is green, but it used to be blue. My favorite book is Catch-22, and my favorite sport is spreadsheets.

What I do

My current focus is on Golang development at USA Today, as part of the team supplying cloud infrastructure services to Gannett media properties across the country. I've previously worked in similar roles, with a focus on cloud deployment and automation, at Target and Amazon Web Services,. I'm knee-deep in Go right now, but I have worked with Coffeescript and Node.js in the past, and enjoyed it far more than I'm told I was supposed to. I'm also familiar with Python and am always up for hammering out something new.

Stuff I've made

Super Tuner for Spotify

Adds a frontend to the advanced search functionality hidden in the Spotify Web API.

(JavaScript. Online at richabdill.com/supertuner, source on GitHub.

Pistol Search

A serverless JavaScript application built using AngularJS plus some elements of material design. Users input criteria into a custom-built faceted search engine that helps wade through handgun design data.

(JavaScript. Currently at pistolsearch.com, source on GitHub.)

Bishop's Map

A browser-based framework for publishing text adventures as web pages or Chrome applications. Allows game authors to define their entire game in JSON and avoid any programming.

(JavaScript, white magic. Under construction. Source available online.)

Worknerd

A project I started to practice scripting and tinker with web spiders. It's a job scraper written in Shell, with a small PHP site that queries the results and displays search data.

(Bash, PHP, MySQL. Source available online.)

Tic-Tac-Toe Robot

A brief venture into machine learning, in which the computer player examines past games of tic-tac-toe to determine (occasionally) correct maneuvers.

(PHP, MySQL.Source available online.)
