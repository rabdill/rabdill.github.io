Who I am

My name is Rich. I ride a motorcycle, I hang out with a cat, a teacup poodle, and a hedgehog named Chalupa. My favorite color is green, but it used to be blue. My favorite book is Catch-22, and my favorite sport is spreadsheets.

What I do

I'm a freshly minted cloud deployment engineer at Target. I like coding and automation. I used to do devops stuff for USA Today and Amazon Web Services. For fun, I write confusing, unsuccessful web applications in the middle of the night. If I could run Jenkins on steam power, I'd do it without a second thought.

Stuff I've made

Bishop's Map

JavaScript, white magic

A browser-based framework for publishing text adventures as web pages or Chrome applications. Allows game authors to define their entire game in JSON and avoid any programming. (Under construction. Source available online.)

Treeclimber

MongoDB, JavaScript (Node.js/AngularJS/Express/Mongoose)

A geneaology tool running very precariously on the MEAN stack. Mostly useful for associating people with the historical documents that support their biographical information. (Under construction. Source available online.)

Worknerd

Bash, PHP, MySQL

A project I started to practice scripting and tinker with web spiders. It's a job scraper written in Shell, with a small PHP site that queries the results and displays search data. (Source available online.)

Tic-Tac-Toe Robot

PHP, MySQL

A brief venture into machine learning, in which the computer player examines past games of tic-tac-toe to determine (occasionally) correct maneuvers. (Source available online.)

Open source contributions

I've played a very minor role in several FOSS projects, mostly in documentation and editing but also several bug fixes. These projects include:

Contacting Congress
A project to standardize a method of contacting members of the U.S. Senate and House of Representatives consisting mostly of YAML files defining the configuration of the congressional websites. I contributed one such file for a member of the House. (More information from the Sunlight Foundation)
Chef Supermarket
Corrected outdated references in the canonical cookbook used to deploy the Opscode Supermarket.
GitHub
Clarified on-site instructions for the GitHub service integration with Trello.
Grails website
Fixed a bug in the (Grails-based) website at Grails.org that was misdirecting users with incorrectly generated links.
DataDog
I was building a custom integration with the popular metric aggregation tool when I found an undocumented API option. I turned it a documented one.
