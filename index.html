Rich Abdill

Maker of small internet things

Who I am

My name is Rich. I'm a Linux sysadmin. I ride a motorcycle, I hang out with a hedgehog named Chalupa. My favorite color is green, but it used to be blue. My favorite book is Catch-22, and my favorite sport is spreadsheets.

What I do

I'm a systems engineer at Gannett, supporting cloud infrastructure behind properties including USA Today, the Arizona Republic, the Indianapolis Star, and WUSA9. My work is almost exclusively within Amazon Web Services, where I assist in the architecture and maintenance of systems starting at DNS and moving through network routing, compute, caching, queuing, and databases. Much of my interest is in automation and continuous integration, and I'm very familiar with systems like Jenkins and Chef. I love writing code – especially when the result is a big, red button that will do my work for me.

Stuff I've made

Bishop's Map

A JavaScript-based framework for publishing text adventures as web pages or Chrome applications. Allows game authors to define their entire game in JSON and avoid any programming. (Under construction. Source available online.)

Worknerd

A project I started to practice scripting and tinker with web spiders. It's a job scraper written in Shell, with a small PHP backend that displays search results. (Source available online.)

Tic-Tac-Toe Robot

A brief, PHP-based venture into machine learning, in which a computer player examines past games of tic-tac-toe to determine correct maneuvers. (Source available online.)
