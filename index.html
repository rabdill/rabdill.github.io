Who I am

My name is Rich. I ride a motorcycle, I hang out with a hedgehog named Chalupa. My favorite color is green, but it used to be blue. My favorite book is Catch-22, and my favorite sport is spreadsheets.

What I do

I'm a consultant with Amazon Web Services, where I work on the architecture and implementation of cloud infrastructure projects, with a focus on helping large companies implement DevOps policies and continuous delivery. For fun, I write confusing, unsuccessful web applications in the middle of the night.

Stuff I've made

Bishop's Map

A JavaScript-based framework for publishing text adventures as web pages or Chrome applications. Allows game authors to define their entire game in JSON and avoid any programming. (Under construction. Source available online.)

Worknerd

A project I started to practice scripting and tinker with web spiders. It's a job scraper written in Shell, with a small PHP backend that displays search results. (Source available online.)

Tic-Tac-Toe Robot

A brief, PHP-based venture into machine learning, in which a computer player examines past games of tic-tac-toe to determine correct maneuvers. (Source available online.)
